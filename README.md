# 💚 Duman IDM (İndirme Yöneticisi)

Python ve Tkinter kullanılarak geliştirilmiş, modern ve kullanıcı dostu bir indirme yöneticisi.

## 🚀 Özellikler
- **Modern Arayüz:** Craveat tasarım dilinden esinlenen mor tema.
- **Dairesel İlerleme:** İndirme durumunu şık bir halka grafik üzerinden takip etme.
- **Kaldığı Yerden Devam:** Bağlantı kopsa bile indirmeye devam edebilme (Resume desteği).
- **Otomatik Adlandırma:** URL'den dosya adını otomatik yakalama.
- **Türkçe Dil Desteği:** Tamamen yerelleştirilmiş arayüz.
